# AI Prompt Library Application

## 🚀 Overview

This is a full-stack application to manage AI Image Generation Prompts. Users can view, create, and track prompt usage.

---

## 🌐 Live Demo

https://design-helper--vmasood02.replit.app

---

## 🛠 Tech Stack

* Frontend: Angular
* Backend: Django
* Database: PostgreSQL
* Cache: Redis
* DevOps: Docker

---

## ✨ Features

* View all prompts
* View prompt details
* Track view count (Redis)
* Add new prompts with validation

---

## 🔗 API Endpoints

* GET /prompts/
* POST /prompts/
* GET /prompts/:id/

---

## ⚙️ Setup Instructions

```bash
docker-compose up --build
```
